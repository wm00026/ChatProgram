# ChatProgram

A basic program that allows user to message each other. The program is based entirely in Java.

## Motive
The motivation for creating this program was to practice Java skills, libraries, and implementation. A chat program, which uses I/O, sockets, executor service, and JUnit tests helps to improve my overall Java skillset.

## Opening the Program:
First, grab all the files. You could clone the repo or download all files (the .zip or .tar). 

Likely, you will have to recompile the build files (the .class files). Do so using 
javac -d (destination) (location of files/*.java)
- Location is likely /src/common

Launch the server side with
java server.ChatServer

Then, launch the client side with:
java client.ChatClient
